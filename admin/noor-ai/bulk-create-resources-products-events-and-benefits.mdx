---
title: "Bulk create resources, products, events, and benefits"
description: "Quickly create or duplicate resources, shop products, events, and benefits to save time on manual data entry with Noor AI."
---

Use Noor AI to handle high-volume content generation with ease.

1. Open [Noor AI](/noor-ai/ask-noor-ai) and start a chat.

2. Tell Noor AI what you want to achieve using one of these methods:

   - **To create from scratch:** Ask Noor AI to create one or multiple resources, shop products, events, or benefits. Specify details like names, prices, visibility, dates, or schedules (e.g., exact dates or “every Tuesday from April to August”).

     <Frame>
       <img
         src="/images/noor-ai/bulk-create-from-scratch.png"
         alt="Creating a new item from scratch using Noor AI"
       />
     </Frame>

   - **To duplicate an existing item:** Create one item manually first (e.g., a desk on the [Resources](/resources) page), then ask Noor AI to duplicate it. Specify how many duplicates you need, their new titles, and whether any specific settings should change or remain identical.

     <Frame>
       <img
         src="/images/noor-ai/duplicate-existing-item.png"
         alt="Duplicating an existing item using Noor AI"
       />
     </Frame>

3. Provide any additional settings or details Noor AI asks you for in the chat.

4. Noor AI will prepare the items and list them for your review. Review each item, then click **Approve** or **Reject**.

   <Frame>
     <img
       src="/images/noor-ai/review-bulk-items.png"
       alt="Reviewing and approving multiple items prepared by Noor AI"
     />
   </Frame>

5. Once confirmed, Noor AI will create all approved items and notify you in the chat.

<Warning>
  You can ask Noor AI to duplicate multiple items at the same time.
</Warning>
