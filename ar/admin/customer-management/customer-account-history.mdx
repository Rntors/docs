---
title: " سجل حساب العميل "
---

يحتوي كل ملف عميل على جدول زمني متسلسل للتغييرات الرئيسية على دور العضوية وإعدادات الفوترة والاشتراكات. يوفر مصدراً موحداً للحقيقة لعلاقاتك مع العملاء.

- جدول زمني كامل لرحلة العميل، من تسجيله الأولي إلى التغييرات في دور عضويته.
- تغييرات إعدادات الفوترة التي تؤثر مباشرة على الرسوم المفروضة على العميل.
- المساءلة وإمكانية التحقق من المسؤول الذي نفّذ إجراءً محدداً، مثل إنشاء أو إلغاء اشتراك العميل، في حالة حدوث أخطاء.

## الوصول إلى سجل حساب العميل

1. انتقل إلى صفحة **Customers** \> علامة التبويب "Members" أو "Nonmembers".
2. افتح ملف المستخدم أو الشركة المطلوبة.
3. تحقق من سجل حساب العميل ضمن حقل "Internal notes".
4. لرؤية التاريخ والوقت الدقيقين لحدث معين، مرّر مؤشر الفأرة فوق الطابع الزمني المجاور له.

## الأحداث الشائعة ومعانيها

### دور العميل

| الحدث | المعنى |
| --- | --- |
| **Customer was created as a nonmember.** | قام المسؤول [بإضافة غير عضو](/admin/customer-management/users/nonmembers). |
| **Customer was created as a member.** | قام المسؤول [بدعوة مستخدم](/admin/customer-management/users/members#invite-a-member) أو [إنشاء شركة](/admin/customer-management/companies/set-up-companies). |
| **Customer was created. \> Customer became a nonmember.** | قام المستخدم بعملية شراء أو حجز وأصبح [غير عضو](/admin/customer-management/users/nonmembers). |
| **Customer was created. \> Customer signed up for a subscription. \> Customer became a member.** | قام المستخدم [بالتسجيل](/admin/billing/plans/immediate-plan-sign-ups) في اشتراك وأصبح [عضواً](/admin/customer-management/users/members). |
| **Customer was created. \> Administrator moved customer to members.** | وافق المسؤول على [طلب الانضمام](/admin/customer-management/users/requests-to-join) للمستخدم أو على دعوة المستخدم للشركة. |
| **Administrator moved customer to members.** | قام المسؤول [بدعوة أو إعادة دعوة غير عضو](/admin/customer-management/users/members) إلى المجتمع أو [استعاد شركة](/admin/customer-management/companies/delete-a-company#restore-a-company). |
| **Administrator moved customer to nonmembers.** | قام المسؤول [بإزالة عضو](/admin/customer-management/users/remove-users) أو [شركة](/admin/customer-management/companies/delete-a-company) من المجتمع. |
| **Customer became a nonmember.** | تمت إزالة الشركة التي ينتمي إليها المستخدم مع أعضائها، لذا أصبح المستخدم تلقائياً غير عضو. |
| **Customer requested to join as a member.** | قدّم المستخدم [طلب انضمام](/admin/customer-management/users/requests-to-join) للموقع كعضو. |
| **Customer canceled the request to join as a member.** | ألغى المستخدم طلب الانضمام المعلّق إلى الموقع كعضو. |
| **Administrator approved the request for customer to join as a member.** | وافق المسؤول على طلب المستخدم للانضمام إلى الموقع كعضو. |
| **Administrator rejected the request for customer to join as a member.** | رفض المسؤول طلب المستخدم للانضمام إلى الموقع كعضو. |

### إدارة الشركة

| الحدث | المعنى |
| --- | --- |
| **Administrator changed the company name.** | قام المسؤول بتعديل اسم الشركة. |
| **Administrator added/removed a user to/from company.** | قام المسؤول بإضافة عضو حالي إلى الشركة أو إزالته منها. |
| **Administrator made a user a company manager/company member.** | غيّر المسؤول دور المستخدم داخل الشركة. |
| **User was added to company.** | <ul><li>سجّل المستخدم في اشتراك شركة وتمت إضافته تلقائياً إلى الشركة كمدير، أو</li><li>قام المسؤول بدعوة المستخدم مباشرة إلى الشركة أو وافق على دعوة المستخدم للشركة.</li></ul> |
| **User was removed from company.** | تمت إزالة الشركة، وتمت إزالة المستخدم تلقائياً منها. |
| **User became a company member.** | تمت إزالة الشركة، وفقد المستخدم تلقائياً دور مدير الشركة فيها. |

### الاشتراك وإعدادات الفوترة

| الحدث | المعنى |
| --- | --- |
| **Customer signed up for a subscription. \> The subscription was created for customer.** | قام المستخدم [بالتسجيل](/admin/billing/plans/immediate-plan-sign-ups) في اشتراك شخصي أو اشتراك شركة. |
| **Administrator created a subscription for customer.** | قام المسؤول [بإنشاء اشتراك](/admin/billing/subscriptions/set-up-subscriptions) لمستخدم أو شركة. |
| **The subscription for customer started.** | بدأ الاشتراك وفقاً لتاريخ البدء المحدد. |
| **Administrator canceled the subscription for customer.** | قام المسؤول [بإلغاء الاشتراك](/admin/billing/subscriptions/cancel-a-subscription). |
| **The subscription was canceled for customer.** | تمت إزالة العميل من المجتمع، وتم إلغاء اشتراكه تلقائياً. |
| **Administrator enabled/disabled Auto-pay for customer.** | غيّر المسؤول [إعداد Auto-pay](/admin/billing/subscriptions/control-invoice-auto-charge) للعميل، مما قد يؤثر على تحصيل الفواتير تلقائياً. |

### حقوق المسؤول

| الحدث | المعنى |
| --- | --- |
| **Administrator made user an administrator/revoked the administrator rights from a user.** | قام المسؤول بتفعيل أو تعطيل [دور "Administrator"](/admin/administration/administrators/administrators) لمستخدم. |
| **Administrator moved a user to nonmembers and revoked the administrator rights from the user.** | قام المسؤول [بإزالة](/admin/customer-management/users/remove-users) مسؤول آخر من المجتمع، وفقد حقوقه كمسؤول تلقائياً. |
| **Customer became a nonmember and lost the administrator rights.** | تمت إزالة الشركة التي ينتمي إليها المسؤول من المجتمع مع أعضائها، لذا فقد المستخدم تلقائياً حقوق المسؤول وأصبح غير عضو. |
