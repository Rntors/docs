## Base URL

```
https://iwdewzswethsnvwvpind.supabase.co/functions/v1/public-api
```

## Authentication

All API requests require two headers:

```bash
Authorization: Bearer YOUR_CLIENT_SECRET
X-Client-ID: YOUR_CLIENT_ID
```

Generate API credentials from **Settings → Developers → API Keys** in your Rntor dashboard.

## Rate Limiting

API requests are rate limited per API key. Default: **60 requests/minute**.

Rate limit headers are included in all responses:

| Header | Description |
|--------|-------------|
| `X-RateLimit-Limit` | Requests allowed per minute |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Unix timestamp when the rate limit resets |

## Response Format

### Success (Single Object)

```json
{
  "data": {
    "id": "abc123",
    "name": "Meeting Room A",
    ...
  },
  "meta": {
    "request_id": "req_abc123def456",
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```

### Success (List with Pagination)

```json
{
  "data": [
    { "id": "abc123", ... },
    { "id": "def456", ... }
  ],
  "pagination": {
    "page": 1,
    "per_page": 20,
    "total": 150,
    "total_pages": 8,
    "has_more": true
  },
  "meta": {
    "request_id": "req_abc123def456",
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```

### Error

```json
{
  "error": {
    "code": "not_found",
    "message": "Booking not found"
  },
  "meta": {
    "request_id": "req_abc123def456"
  }
}
```

## Error Codes

| Code | HTTP Status | Description |
|------|-------------|-------------|
| `unauthorized` | 401 | Missing or invalid API credentials |
| `forbidden` | 403 | Valid credentials but insufficient scope |
| `not_found` | 404 | Resource does not exist |
| `validation_error` | 400 | Invalid request parameters |
| `rate_limited` | 429 | Rate limit exceeded |
| `internal_error` | 500 | Server error |

## Pagination

List endpoints support pagination via query parameters:

| Parameter | Default | Max | Description |
|-----------|---------|-----|-------------|
| `page` | 1 | - | Page number |
| `per_page` | 20 | 100 | Items per page |

## Example Request

```bash
curl -X GET "https://iwdewzswethsnvwvpind.supabase.co/functions/v1/public-api/bookings?status=confirmed&per_page=10" \
```
